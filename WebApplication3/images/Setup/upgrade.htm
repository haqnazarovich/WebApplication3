Thank you for using the trial edition of Microsoft Visual Studio 2010

Trial editions are valid for 30 days.  You can extend the trial period for Visual Studio 2010 to 90 days by registering online or convert the installation to a fully licensed version with a valid product key.  The retail product key can be found on the DVD sleeve of the installation media.

Please note: Pre-release versions of Visual Studio 2010 cannot be converted to fully licensed version.  You must uninstall all Visual Studio 2010 pre-releases before installing the official release.

Buying Visual Studio 2010

To purchase Visual Studio 2010, either buy the product at a retail outlet, or order a product key online by clicking here.

Activate the product

You can activate the product in one of the following ways:
  • From Microsoft Visual Studio 2010 Setup Maintenance page
    1. Click Run from Start menu (Start+R) and proceed to step 2, or run setup.exe from your Visual Studio installation media and proceed to step 5.
    2. Type control appwiz.cpl, and press OK.
    3. Select Microsoft Visual Studio 2010 (Professional, Premium, Ultimate, etc.).
    4. Click Change to launch Visual Studio 2010 setup in maintenance mode.
    5. Click Activate.
  • Microsoft Visual Studio 2010 Help About page
    1. Launch Microsoft Visual Studio 2010.
    2. Click About Microsoft Visual Studio from the product's Help menu.
    3. Click Upgrade button to launch Microsoft Visual Studio 2010 Activation dialog.
  • Microsoft Visual Studio 2010 Activation dialog
    1. Type the new valid key in the Product Key prompt.
    2. Click Activate.
