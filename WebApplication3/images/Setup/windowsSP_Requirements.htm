Microsoft Visual Studio 2010

Supported Operating Systems Service Levels:

  • Windows XP Service Pack 3
  • Windows Server 2003 Service Pack 2
  • Windows Vista Service Pack 2
  • Windows Server 2008 Service Pack 2
  • Windows Server 2008 R2
  • Windows 7

Visual Studio requires that minimum supported Windows Service Pack is installed on your computer before you install Visual Studio.  You can download and install Windows service pack from http://windowsupdate.microsoft.com..  After you install the service pack, you are required to restart your computer.  Visual Studio setup will not automatically restart.  To restart Visual Studio setup, double-click setup.exe from the original installation media or click the appropriate link on the download Web site.

Note:  It is highly recommended that you regularly check for and install updates for your operating system available at http://windowsupdate.microsoft.com/a>..

Why install Service Packs?

Service packs are the means by which product updates are distributed.  Service packs can contain updates to improve system reliability, program compatibility, security, and more.  For more information on Microsoft Service Packs, see http://support.microsoft.com/efault.aspx?scid=fh;EN-US;sp.
